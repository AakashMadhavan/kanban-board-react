# Kanban Board (React)

A clean and responsive Kanban Board application built using React.

## 🚀 Features

* Drag and drop tasks between columns
* Add, edit, and delete tasks
* Multiple columns (To Do, In Progress, Done)
* Clean and intuitive UI

## 🛠 Tech Stack

* React (Functional Components + Hooks)
* Context API (State Management)
* @hello-pangea/dnd (Drag & Drop)
* Vite

## 📦 Installation

```bash
npm install
npm run dev
```

## 📂 Project Structure

* components/ → UI components (Board, Column, TaskCard)
* context/ → Global state management
* data/ → Initial data

## 🎯 Purpose

This project was built to demonstrate React skills, component design, and drag-and-drop functionality.
